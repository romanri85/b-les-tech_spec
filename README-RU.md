# b-les-tech_spec

Техническая спецификация для проекта b-les

# 1. Цели проекта

Цели проекта: разработать сайт для компании, которая производит и продает (B2C - розничная торговля) межкомнатные
двери в Москве, Санкт-Петербурге и других городах России. Продукция компании: межкомнатные двери, дверные ручки и
другая дверная фурнитура. Услуги компании: доставка, установка, замеры, консультации и т.д. Посетитель может
прочитать о компании, ее продукции и услугах.

# 2. Функциональность сайта

Сайт должен содержать следующие страницы:

1. Страница "Главная"
2. Страница "Каталог"
3. Страница "Где купить"
4. Cтраница "Двери в интерьере"
5. Страница "Фурнитура"
6. Страница "О компании"
7. Станицу "Избранное"
8. Страница "Дизайнерам"
9. Страница "Системы открываний"
10. Страница "Акции"
11. Страница "Установка и доставка"
12. Блок хедера
13. Блок футера

## 2.3. Страница "Где купить"

### 2.3.1. Города на странице "Где купить"

Когда посетитель заходит на любую страницу сайта, система предлагает ему подтвердить свое местоположение с помощью
всплывающего окна. Внешняя SaaS получает IP-адрес пользователя и определяет его местоположение по умолчанию. Если
посетитель соглашается, система сохраняет его местоположение в локальном хранилище браузера, если нет: пользователь
может выбрать местоположение из доступных вариантов (городов), где расположены магазины. Посетитель может изменить свое
местоположение в любое время. В верхней части страницы есть иконка "Мой город". Каждый город, в котором есть магазин
компании, должен иметь url с картой Google, где расположены магазины.

### 2.3.3. Магазины на странице "Где купить"

В каждом городе, где есть магазины компании, должен быть список магазинов с их адресами,
телефонами и временем работы. Существует 2 типа магазинов: монобрендовый магазин (в нем продается только продукция
компании) и
мультибрендовый магазин, где также продаются товары других компаний. Тип магазина должен быть виден в информационном
блоке данного магазина. Каждый магазин имеет адрес, номер телефона, часы работы, тип. У каждого магазина есть
Оставшееся время работы каждого магазина должно отображаться, оно должно обновляться
каждую минуту. Это время вычисляется динамически на стороне клиента в соответствии с часовым поясом города, который
пользователь
выбранного пользователем в качестве местного города. Если пользователь не выбрал ни одного магазина, то используется
часовой пояс, определяемый IP-адресом пользователя.
Если IP-адрес пользователя не определен, часовой пояс должен быть взят из браузера.

Маршрут до магазина может быть рассчитан пользователем с помощью Google Maps. Пользователь может выбрать магазин из
списка
и нажать на кнопку "Рассчитать маршрут". Маршрут будет рассчитан и отображен на карте. В случае если браузер
пользователя не позволяет определить его местоположение, система должна попытаться получить это разрешение от браузера.
Если
разрешение не получено, система должна вывести сообщение "Пожалуйста, разрешите определить ваше местоположение для
расчета
маршрут".

### 2.8.2. Функциональность страницы "Дверная фурнитура"

Страница должна представлять дверную фурнитуру так же, как страница "каталог" представляет двери.
Дверная фурнитура должна быть разделена на несколько типов:

1. Дверные ручки
2. Дверные петли

Страница должна иметь следующие фильтры:

1. Тип фурнитуры
2. Цвет
3. Коллекция
4. Дизайн
5. Ценовой диапазон

В верхней части страницы должен быть блок с наиболее популярными наборами фильтров. Пользователь может нажать на любой
набор фильтров
и система должна применить этот набор фильтров к странице.

Существует два типа дверной фурнитуры, до 20 цветов, до 10 коллекций, 3 типа дизайна.
Каждый тип дверной фурнитуры может иметь несколько цветов, несколько коллекций, несколько типов дизайна.
Каждая коллекция может иметь несколько цветов, только 1 тип дизайна, только 1 тип дверной фурнитуры.
Каждый тип дизайна может иметь несколько коллекций, несколько цветов, несколько типов дверной фурнитуры.
Каждый цвет может иметь несколько коллекций, несколько типов дизайна, несколько типов дверной фурнитуры.
Каждый элемент дверной фурнитуры может иметь только 1 коллекцию, только 1 тип дизайна, только 1 тип фурнитуры, несколько
цветов.

Каждый элемент дверной фурнитуры, представленный на странице, должен содержать следующую информацию:

1. Название
2. Изображение
3. Цена
4. Коллекция
5. Цвет

Пользователь может нажать на любой элемент дверной фурнитуры и увидеть увеличенное изображение со всеми тегами,
прикрепленными к этому изображению. На сайте
пользователь может нажать на любой тег, при этом он будет перенаправлен на страницу "Двери в интерьере" в режиме "Двери
в интерьере".
Она должна открыться в новой вкладке браузера. Пользователь должен иметь возможность вернуться на страницу "Дверная
фурнитура" путем
вернувшись на предыдущую вкладку.
